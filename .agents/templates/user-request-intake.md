# 用户需求录入模板

> Agent 收到用户消息后，先按此表整理（未涉及区块填 **N/A**），再开工。整理结果可追加到 `.agents/facts/user-request-log.md` 置顶。

| 区块 | 内容 |
|------|------|
| **新增逻辑** | （功能点列表；无则 N/A） |
| **Bug 检查** | （现象 + 期望；无则 N/A） |
| **架构调整** | （`.agents/`、构建、数据层等；无则 N/A） |
| **其他** | （普通对话、说明、非开发项；无则 N/A） |

## 示例

| 区块 | 内容 |
|------|------|
| **新增逻辑** | 日历：非当月自动选最早有账日；标题点选年月。账户：活期/资产/全部筛选，新增后切到对应类型 |
| **Bug 检查** | 日历切回当月偶发未选中今天 |
| **架构调整** | `.agents/` 用户输入自动整理为本模板，空缺填 N/A |
| **其他** | N/A |
